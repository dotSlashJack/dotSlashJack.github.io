Gephi to vis.js tool

coded by Jack Hester, hosted by google and github

Convert Gephi network graph to a web-friendly network graph

Download Here

This python tool converts network diagrams made in Gephi to a web friendly format powered by vis.js
Usage
From gephi:
   1) Open the data tables (window > data table).
   2) Click "nodes" and click "export table" to save nodes as a CSV.
   3) Click "edges" and click "export table" to save edges as a CSV.
   4) Run the python tool. Use the -h flag for details. It is recommended to place the python file and CSVs in a dedicated directory.
Note: you will need vis.min.js, vis.min.css, and a template file. These will be downloaded automatically for you.
